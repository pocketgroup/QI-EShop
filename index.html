This is a demo website of a fictional ACME e-shop.
The main purpose of this app is to demonstrate the way how to dynamically load content within an existing page.

Catalog page content is dynamically loaded from a different web site maintained by a 3rd party company. However the links from catalog lead back to the Acme e-shop.
Product page shows a list of products. It's maintained by the Acme company.
